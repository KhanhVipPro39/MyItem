##  Command:

<br>

**All MyItem Commands:**

| **Command** | **Description** |
| --- | --- |
| **/setname** | **Command help edit item name**
| **/setlore** | **Command help edit item lore (use \\n for newline)**
| **/addench** | **Command help add enchant to item**
| **/listench** | **Command help open list enchant**
